\section{Title Page Setup}
After inserting the package, you should define the title and author name. Here is an example, which is the code of this demo:
\begin{verbatim}
    \documentclass{article}
    \input{hassium.tex}
    \begin{document}
        \def\htitle{Demo of Hassium Style}
        \def\hauthor{Hassium}
        \def\hfauthor{Hassium}
        \hsetup
        \htoc
        \hmain
    \end{document}
\end{verbatim}
The command ``hsetup'' gives you the title and the author name. The command ``\texttt{htoc}'' gives a table of contents, which we will mention later. The command ``\texttt{hmain}'' is a setup of the mainmatter, which includes a fancy header. The ``\texttt{hfauthor}'' variable is the left part of the header. Also, feel free to use ``\texttt{hstart}'' command to include all three of them.
\begin{verbatim}
    \documentclass{article}
    \input{hassium.tex}
    \begin{document}
        \def\htitle{Demo of Hassium Style}
        \def\hauthor{Hassium}
        \def\hfauthor{Hassium}
        \hstart
    \end{document}
\end{verbatim}

\section{More on Table of Contents}
You can add descriptions to each section and the description will appear in the table of contents, directly below the section name: 
\begin{verbatim}
    \section{This is a Sample Section} 
    \descr{This is a description to the section} 
\end{verbatim}
The table of contents only shows the section names, but no subsections and numberless sections. If you want a numberless section in the table of contents, use the ``newsection'' command:
\begin{verbatim}
    \newsection{This is a numberless section} 
\end{verbatim}
Note that the section names in the table of contents are hyperlinks; click on any section name to navigate directly to that section. You can do the converse to navigate to the first page as well.
\section{Index Page}
This style has a customized index page. Check the code:
\begin{verbatim}
    This is a \hdef{defintiion}. This is another \hdef{vocabulary}.
    \hindex
\end{verbatim}
The command ``hdef'' mark the word and print it. The command ``hindex'' is a customized index page that print words in three columns. Each page number in the index page contains a hyperlink to that page.
\section{Darkmode}
Darkmode commands change the background color to black and the text to white. Similar to the normal setup, there are darkmode setup:
\begin{verbatim}
        \darkhsetup
        \darkhtoc
        \darkhmain
\end{verbatim}
\section{Other Environments and Commands}
The line-spacing in ``\texttt{enumerate}'' environment is changed:
\begin{verbatim}
    \setlist[enumerate]{topsep=0pt,itemsep=-1ex,partopsep=1ex,parsep=1ex}
\end{verbatim}
The ``\texttt{level}'' environment is used in ``\texttt{enumerate}'' environment, consider the following code:
\begin{verbatim}
    \begin{enumerate}
        \item This is the first line.
        \begin{level}
            \item This is the second line.
            \begin{level}
                \item This is the third line.
            \end{level}
            \item This is another line.
        \end{level}
    \end{enumerate}
\end{verbatim}
This code gives:
\begin{enumerate}
    \item This is the first line.
    \begin{level}
        \item This is the second line.
        \begin{level}
            \item This is the third line.
        \end{level}
        \item This is another line.
    \end{level}
\end{enumerate}
The command ``\texttt{circled}'' draws a small circle and you can add something inside the circle:
\begin{verbatim}
    \circled{1}
\end{verbatim}
The output is $\circled{1}$. You can write any Romam numerals by:
\begin{verbatim}
    \rom108 
\end{verbatim}
There are two simple commands for hand-written fonts:
\begin{verbatim}
    \cfd{font 1}
    \cfc{font 2}
\end{verbatim}
The outputs are \cfd{font 1} and \cfc{font 2}.
\section{Quiver}
Quiver is done by \href{https://github.com/varkor}{varkor} and \href{https://tex.stackexchange.com/users/138900/andr%C3%A9c}{Andr√©C}, check their github for more information. I include quiver to draw curve arrows in a commutative diagram. To draw a diagram with quiver, check this \href{https://q.uiver.app/}{website}. An example is given below:
\begin{verbatim}
    \begin{center}
        \begin{tikzcd}
            Hello &&&& World \\
            \\
            \\
            && Hassium
            \arrow["\shortmid"{marking}, curve={height=-6pt}, tail reversed, from=1-1, to=1-5]
            \arrow[curve={height=6pt}, squiggly, from=1-1, to=4-3]
            \arrow[curve={height=-6pt}, dashed, hook', from=1-5, to=4-3]
        \end{tikzcd}
    \end{center}
\end{verbatim}
The diagram looks like:
\begin{center}
    $\begin{tikzcd}
        Hello &&&& World \\
        \\
        \\
        && Hassium
        \arrow["\shortmid"{marking}, curve={height=-6pt}, tail reversed, from=1-1, to=1-5]
        \arrow[curve={height=6pt}, squiggly, from=1-1, to=4-3]
        \arrow[curve={height=-6pt}, dashed, hook', from=1-5, to=4-3]
    \end{tikzcd}$
\end{center}
\section{Theorem Styles}
Several theorem styles are offered:
\begin{verbatim}
    \theoremstyle{definition}
    \newtheorem{definition}{Definition}[section]
    \newtheorem{theorem}{Theorem}[section]
    \newtheorem*{proposition}{Proposition}
    \newtheorem*{lemma}{Lemma}
    \newtheorem*{corollary}{Corollary}
    \newtheorem*{example}{Example}
    \newtheorem*{remark}{Remark}
    \newtheorem*{notation}{Notation}
    \newtheorem{problem}{Problem}[section]
    \newtheorem*{claim}{Claim}
\end{verbatim}
The environment name can be customized by using:
\begin{verbatim}
    \customtheorem{This is a custom theorem}
    \begin{This is a custom theorem}
        The proof is trivial.
    \end{This is a custom theorem}
\end{verbatim}
The output will be:
\customtheorem{This is a custom theorem}
\begin{This is a custom theorem}
    The proof is trivial.
\end{This is a custom theorem}
\noindent If you don't want to include section number but still want to have a counter in a single section, please use:
\begin{verbatim}
    \begin{adefinition}
        A definition in appendix.
    \end{adefinition}
    \begin{atheorem}
        A theorem in appendix.
    \end{atheorem}
\end{verbatim}
You can put any number or label in ``\texttt{exercise}'' environment:
\begin{verbatim}
    \begin{exercise}[8.6]
        The proof is trivial.
    \end{exercise}
\end{verbatim}
The environment looks like:
\begin{exercise}[8.6]
    The proof is trivial.
\end{exercise}
\section{Invisible Proofs}
The environment ``\texttt{reviewmode}'' is originally done by my friend \href{https://github.com/ETwilight}{ETwilight}. It replaces your ``proof'' environment by three empty lines:
\begin{verbatim}
    \begin{reviewmode}
        \begin{proof}
            The proof is trivial.
        \end{proof}
    \end{reviewmode}
\end{verbatim}
\section{Drawing Functions}
The function drawing command is based on the package \texttt{pgfplots}, you may use the following code:
\begin{verbatim}
    \functiondraw{3}{0.8}{-5}{5}{exp(-x^2-y^2)*x}
    \functiondraw{2}{0.8}{-5}{5}{exp(x^-2*sin(deg(x)))}
    \functiondraw{3}{0.8}{-5}{5}{sin(deg(sqrt(x^2+y^2)))/sqrt(x^2+y^2)}
\end{verbatim}
The first parameter decides the dimension of the graph. If you want the graph to be 2D, then set 2 to the first parameter. Otherwise, give a random integer that does not equal to 2 would give you a 3D graph. The second parameter is the scale of your plot. The third and the fourth is the domain of the plot, in our example, the domain is $[-5,5]$. The last entry is for your function. The example code (with centering) yields the following plot.
\begin{center}
    \functiondraw{3}{0.8}{-5}{5}{exp(-x^2-y^2)*x}
    \functiondraw{2}{0.8}{-5}{5}{exp(x^-2*sin(deg(x)))}
    \functiondraw{3}{0.8}{-5}{5}{sin(deg(sqrt(x^2+y^2)))/sqrt(x^2+y^2)}
\end{center}
